Things I've built, contributed to, or helped bring into the world.

## Software

**[Scriptty](https://github.com/stultus/scriptty)** — An offline desktop screenwriting app for Malayalam and English. Hollywood and Indian two-column PDF export, Mozhi/Inscript input, scene navigator. Built with Tauri 2, SvelteKit, and Typst.

**[PeARS Federated](https://github.com/PeARSearch/PeARS-federated)** — The federated version of [PeARS](https://pearsproject.org/), a community-driven distributed search engine. In this mode, admins create instances and users contribute to the index. I'm a contributor.

**[mp4-to-mpegdash-py](https://github.com/stultus/mp4-to-mpegdash-py)** — Python script to convert MP4 files into on-demand MPEG-DASH format for adaptive streaming.

**[3in1-Mal-Input](https://github.com/stultus/3in1-Mal-Input)** — Combined Mozhi, Swanalekha, and Inscript Malayalam input method JavaScript libraries in one package.

**[Aksharashala](https://github.com/smc/font-installer)** — A font installer and updater for SMC Malayalam fonts on Windows and macOS. Install, preview, and keep all SMC fonts up to date with a single click.

**[Flask-WADO](https://github.com/stultus/Flask-wado)** — A WADO (Web Access to DICOM Objects) server written in Python using Flask. For medical imaging systems.

**[hugo-digital-garden-theme](https://github.com/stultus/hugo-digital-garden-theme)** — Hugo theme for digital gardens. Interactive network graph, note maturity stages, search, and fullscreen mode.

**[stupidly-minimal](https://github.com/stultus/hugo-stupidly-minimal)** — A clean, monospace Hugo theme for personal websites. Dark mode, responsive, Malayalam support.

**[mathml2tex](https://github.com/stultus/mathml2tex)** — XSLT 1.0 transformation from MathML to LaTeX.

**[DicomParser](https://github.com/stultus/DicomParser)** — Parse a DICOM file and return it as a Python dictionary.

**[Chess Mate](https://github.com/stultus/chess-mate)** — A chess playing companion with built-in move analysis. Play against a friend (or yourself) while getting real-time move suggestions powered by a client-side chess engine.
